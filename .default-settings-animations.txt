Disable Window Animation
defaults write NSGlobalDomain NSAutomaticWindowAnimationsEnabled -bool NO

Speed Up Mission Control Animation - default 0.75
defaults write com.apple.dock expose-animation-duration -float 0.15

Disable resize window animation
defaults write -g NSWindowResizeTime -float 0.003

Set an Ultra-Fast Dock Hide & Display Animation
defaults write com.apple.dock autohide-time-modifier -float 0.30;killall Dock
disable: defaults delete com.apple.dock autohide-time-modifier;killall Dock
